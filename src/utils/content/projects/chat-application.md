# Meta Messenger App Micro Clone

**Duration:** Sep 2023 – Sep 2023

## Overview

The **Meta Messenger App Micro Clone** is a sleek and functional messaging application that supports both one-on-one and group chats. Users can log in securely using their Google accounts, join group chats, and enjoy a seamless real-time chatting experience.

## Key Features

- **Group Chat:** Enables one-on-one and group conversations.
- **Google Authentication:** Secure login using Google accounts through Next-Auth.
- **Real-Time Messaging:** Optimistic message updates using Pusher.
- **Modern UI:** Styled with Tailwind CSS for a clean, responsive interface.
- **Human-Readable Time Stamps:** Uses `react-time-ago` for intuitive message history.
- **Optimized Data Fetching:** Efficient data handling with SWR hooks.

## Technologies Used

- **Full Stack Framework:** `Next.js 13.4` with `React` – for both front-end and back-end capabilities.
- **Authentication:** `Next-Auth` with Google provider – for secure user authentication.
- **Real-Time Updates:** `Pusher` – for instant message delivery and updates.
- **Styling:** `Tailwind CSS` – utility-first CSS for rapid UI development.
- **Database:** `Upstash` – serverless Redis for scalable data storage.
- **Icons:** `Heroicons` – beautiful and consistent icon set.
- **Time Display:** `react-time-ago` – for human-friendly timestamps.
- **Data Fetching:** `SWR` – efficient client-side data fetching and caching.

## Summary

This micro clone of a messenger app effectively leverages modern full-stack technologies to deliver a secure, real-time chatting experience. It is ideal for fast, interactive communication in both one-on-one and group settings.
